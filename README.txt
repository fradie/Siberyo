"Baralho" apresenta uma biblioteca de manipulação de um baralho composto de 52 cartas, sem curingas, fornecendo uma representação das cartas e métodos para manipulação das mesmas.

Sobre a construção:

Foi utilizado o "Apache Ant" para construção.
Para executar, digite no terminal: ant


O jogador pode optar entre as funções de:
 
a) embaralhar;
b) cortar em duas partes (em uma determinada posição e juntando as partes ao final);
c) retirar uma carta do inicio e do final;
d) passar uma carta do inicio para o final do monte; 
e) criação de um monte de descarte.
